# Carousel with buttons and page indicator

## use [CRA](https://github.com/facebook/create-react-app) to create the app
```
npx create-react-app carousel
cd carousel
yarn start
```

## auto slide show
study this [pen](https://codepen.io/nashio/pen/xLKepZ).
```
<AutoSlide pics={pics} />
```
